Technologies web

Il existe un certain nombre de technologies web. Cependant, dans le cadre de ma formation, j’ai surtout eu affaire aux suivantes : HTML & CSS pour la mise en forme, JavaScript, PHP pour transmettre les informations entrées par le client au serveur et les réponses du serveur au client et enfin Java pour créer des serveurs Apache. Nous avons également vu une technologie liant la mise en forme et la liaison avec serveur dans le cas d’Angular, mêlant HTML, CSS et TypeScript, un langage proche du JavaScript.

Nous avons utilisé ces technologies tout au long de notre scolarité, réalisant divers projets, allant du simple site vitrine pour une association locale, à un site permettant de demander et de dispenser des formations en passant par un blog dispensant des formations sur les outils de collaboration en ligne.
